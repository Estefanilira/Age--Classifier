## Project Description: "Age Classifier"

This JavaScript project is designed to engage with the user, calculate their age, and categorize them based on their birth year and gender. The user interface includes the following features:

PT BR: 
Este projeto em JavaScript foi desenvolvido para interagir com o usuário, calcular sua idade e categorizá-lo com base no ano de nascimento e gênero fornecidos. A interface do usuário inclui os seguintes recursos:

## Goal / Objetivo

* Input Fields: The program prompts the user to enter their birth year and select their gender from options like male or female.

* Age Calculation: Upon entering the birth year and selecting the gender, the program calculates the user's age based on the current year.

* Age Category: The program categorizes the user as a child, young, adult, or senior based on their age and provides a message like "You are a young adult."

* Gender Indication: It indicates the user's gender, such as "male" or "female," and displays a corresponding image.

* Buttons: There are two buttons available - one for verifying the age and another for resetting the input fields.

Pt-BR
* Campos de Entrada: O programa solicita ao usuário que insira o ano de nascimento e selecione o gênero entre opções como masculino ou feminino.

* Cálculo da Idade: Após inserir o ano de nascimento e selecionar o gênero, o programa calcula a idade do usuário com base no ano atual.

* Categoria de Idade: O programa categoriza o usuário como criança, jovem, adulto ou idoso com base em sua idade e fornece uma mensagem como "Você é um adulto jovem".

* Indicação de Gênero: Ele indica o gênero do usuário, como "masculino" ou "feminino", e exibe uma imagem correspondente.

* Botões: Existem dois botões disponíveis - um para verificar a idade e outro para redefinir os campos de entrada.

##

To implement this project, you can use JavaScript to capture user input, perform age calculations, display messages, and change images dynamically based on the user's selected gender. You'll also need to set up event listeners for the buttons to trigger the age verification and reset functionality.

This project provides an interactive and engaging way for users to determine their age category and gender with a fun and informative interface.
